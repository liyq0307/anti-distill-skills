# /anti-distill-decoy

Generate an outward-facing decoy package.

## Usage

`/anti-distill-decoy <input_path> [output_dir]`

Defaults:
- `output_dir`: `out-decoy`

## Execution

```powershell
python .\anti-distill\scripts\anti_distill.py generate-decoy --input <input_path> --output-dir <output_dir>
```
