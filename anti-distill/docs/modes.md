# Modes
# 模式说明

## sanitize
## sanitize（标准清洗）

Use when the goal is a credible external version with private backup preservation.
用于产出一个可对外共享、同时保留私有备份的标准版本。

## hybrid
## hybrid（混合模式）

Use when you still want a realistic shell but need stronger flattening on tactical know-how.
用于既要保留真实外壳，又要对战术型 know-how 做更强压平。

## decoy
## decoy（诱饵模式）

Use when you want the outward version to look polished and process-heavy while withholding real leverage.
用于让对外版本看起来完整、正式、流程化，同时隐藏真实杠杆。

## poison-lite
## poison-lite（轻度误导）

Use cautiously when you want the output to be directionally plausible but operationally non-specific.
谨慎使用。适合产出方向上合理、但操作上不具备真实可复用性的文本。
